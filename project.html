Profile Project Reference Assignment

สติ๊กเกอร์ไลน์นี้เป็นผลงานชิ้นแรกจากวิชาอนิเมชันที่ผ่านมา สามารถทำเงินให้ได้และเพิ่มทักษะการวาดรูปของตัวเองให้ดีขึ้น ที่ทำรูปหมีเพราะหมีน่ารัก ตุ้ยนุ้ย น่าฟัด และคำต่าง ๆ ที่อยู่ตามสติ้๊กเกอร์คือคำที่ใช้อยู่ในชีวิตประจำวัน

ผลงานนี้เป็นส่วนประกอบของงานทำอนิเมชันการ์ตูนหนึ่งนาที ซึ่งเป็นงานที่ต้องวาดมือเองทั้งหมด ทำด้วยฝีมือของตัวเอง วาดฉากเอง วาดตัวละครหลักเอง วาดตัวประกอบเอง และตัดคลิปเองทั้งหมด ผลงานชิ้นนี้ถือว่าท้าทายความสามารถของตัวเองอย่างมาก เพราะต้องใช้ฝีมือในการวาดรูปและการตัดเสียง ตัดคลิป หรือการทำอนิเมชันเคลืื่อนไหวผ่านโปรแกรม

ภาพนี้จะเป็นการทำแลปต่อวงจร เป็นการเริ่มต่อวงจร และเขียนโค้ดวงจรครั้งแรกของตัวเอง เป็นครั้งแรกที่ทำให้เปิดใจกับโค้ดมากขึ้น เพราะทำเองได้ พอเข้าใจหลักการ ระหว่างเรียนก็ต่อวงจรนี้ไปเรื่อย ๆ ตามแต่ละโจทย์ ก่อนจบปีการศึกษานั้นต้องมีการทำโปรเจกต์ก่อนจะจบ แต่แล้วโปรเจกต์นั้นก็ผ่านไปได้ด้วยดี ไม่มีปัญหาติดิดขัดอะไร
